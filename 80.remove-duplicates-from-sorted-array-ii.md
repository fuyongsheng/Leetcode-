# 80.Remove Duplicates from Sorted Array II \#

![](.gitbook/assets/image%20%284%29.png)

给出一个已排序数组，移除其中重复的多余数字。每个数字最多允许出现两次。

## 方法一：

```java
class Solution {
    public int removeDuplicates(int[] nums) {
        int i = 0;
        for (int n : nums)
            if (i < 2 || n > nums[i-2])
                nums[i++] = n;
        return i;
    }
}
```

**时间复杂度\(Time Complexity\) :** O\(n\)          **空间复杂度\(Space Complexity\):** O\(1\)
